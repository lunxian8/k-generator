# lunxian-generator

#### 项目介绍
代码生成器 根据xxl-gen和人人代码生成器修改支持代码预览打包等等

#### 软件架构
软件架构说明
1. spring boot 2.x 
2. Mybatis Plus 2.x
#### 安装教程

1. 导入SQL
2. 修改数据库连接
3. 运行LunXianApplication类
4. http://127.0.0.1:547
